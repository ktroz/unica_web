Etapas para formarte como becario de UNICA

Primera fase

En esta fase se recibe capacitación sobre temas de cómputo y las relaciones humanas con duración de un semestre escolar. Al final se realiza una evaluación y selecció para pasar a la segunda fase. Cosas que se aprenderán en el curso:

  • HTML
  • Lenguaje C
  • Java
  • REdes de Datos
  • Formación de instructores
  • Linux básico
  • Administración de servidores
  • Python
  • Seguridad en cómputo
  • Base de datos

Segunda fase

En esta fase se pone en practica todo lo aprendido en la primera fase, realizando prácticas durante un semestre en los departamentos que conforman a UNICA. Al finalizar se hará una nueva evaluación y selección para la tercera fase.

Tercera fase

Esta es la última fase, te conviertes en becario y puedes desarrollarte profesionalmente en uno de los departamentos de UNICA y especializarte en algún area de cómputo. Esta fase consta de tres etapas, a las cuales podrás escalar de acuerdo a tu desempeño. Cada una tiene una duración de un semestre.-
